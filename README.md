# 消費アトラス — Shohi Atlas

政府統計をもとに、消費に関するデータを地図で眺めるためのサイトです。
日本(都道府県別)と世界(国別)の2つのページで構成されています。

## 日本ページ (index.html)

全国家計構造調査 (e-Stat) をもとに、都道府県別の1世帯当たり消費支出を
コロプレスマップ(塗り分け地図)で表示します。

- **指標の切り替え**: 消費支出などの基本指標をワンクリックで切り替え
- **属性で見る**: 世帯主の年齢階級 (29歳以下〜70歳以上) を選ぶと、
  そのセグメントの消費額で地図全体を塗り分け
- **2カテゴリ比較**: 例えば「30代 − 60代」のような差分マップを表示。
  差がプラスの県は藍、マイナスの県は朱のダイバージング配色
- **年スライダー**: 2014年・2019年・2024年の3時点を切り替えて時系列比較
- **詳細パネル**: 都道府県をクリックすると、各指標の値・全国順位・
  年齢階級別の内訳グラフを表示
- **ランキング**: 現在の表示条件に連動した順位表。TOP10と全47都道府県の
  一覧を切り替え可能

## 世界ページ (world.html)

World Bank Open Data の家計最終消費支出をもとに、国別の消費を表示します。

- 1人当たり家計消費 (USD)・総額・人口の切り替え(分布の偏りを考慮した対数スケール)
- 年スライダーによる時系列比較
- 国をクリックすると詳細と1人当たり消費の推移を表示
- TOP10 / 全か国のランキング切り替え

## データについて

表示データは `data/` 内の静的な JSON です。サイト自体が外部 API を呼ぶことは
ありません。データの再生成には `scripts/` のスクリプトを使用します
(日本: e-Stat API、世界: World Bank API)。

年齢階級の属性は5歳刻みの原数値を単純平均で6区分に集約しています。また
2014年の属性は前身調査(全国消費実態調査)由来のため、2019年以降と調査設計が
一部異なります。

## 出典

- 統計データ: 政府統計の総合窓口 (e-Stat)「全国家計構造調査」を加工して作成
- 世界データ: World Bank Open Data (CC BY 4.0)
- 日本地図: 国土交通省 国土数値情報 (行政区域データ) を加工
- 世界地図: Natural Earth (パブリックドメイン) / world-atlas
